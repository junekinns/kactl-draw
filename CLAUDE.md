# 웹 기반 랜덤 숫자뽑기 프로그램

번호추첨 행사에 쓰일 랜덤 숫자뽑기 프로그램.
대형 스크린에서 사용되며, 시각적 연출이 중요하다.

## 핵심 기능
1. 시작숫자, 끝숫자 범위 지정
2. 뽑을 갯수 지정
3. 제외할 숫자 지정 (콤마 구분)
4. 추첨 버튼을 누르면 애니메이션과 함께 결과 표시

## 기술 스택
- 프론트엔드: HTML/CSS/JavaScript (단일 페이지 웹앱)
- 배포: GitHub → GCP (GitHub Actions CI/CD)

## Agents

### 1. graphic-designer (그래픽 디자이너)
- 로또 공 스타일 숫자 볼, 추첨 애니메이션, 전체 페이지 레이아웃, 색상, 타이포그래피 설계
- 대형 스크린에서 잘 보이는 큰 글씨와 화려한 시각 효과 우선
- CSS 애니메이션, SVG, Canvas, confetti/파티클 축하 이펙트 구현
- 반응형 디자인 적용

### 2. frontend-developer (프론트엔드 개발자)
- 핵심 로직 구현: 범위 입력, 갯수 입력, 제외 숫자 입력, 중복 없는 랜덤 추첨
- 입력값 유효성 검증 (범위 오류, 뽑을 수 초과 등)
- 디자이너가 만든 시각 요소 통합

### 3. cicd-engineer (CI/CD 엔지니어)
- GitHub Actions 워크플로우 작성
- GCP (Cloud Run 또는 Firebase Hosting) 자동 배포
- main 브랜치 푸시 시 프로덕션 배포

### 4. qa-engineer (QA 엔지니어)
- 범위 내 랜덤 숫자 정확 생성 검증
- 제외 숫자 미포함 검증
- 중복 없이 뽑히는지 검증
- 경계값 처리 (시작=끝, 뽑을 수=가능 수 전체 등)
- 잘못된 입력에 대한 에러 처리 검증
- 브라우저 호환성, 대형 스크린 표시 점검
